Hacked Hogwarts

This project was my first solo project on the frontend elective.

The assignment was to build a system to help the administrators at Hogwarts handle student lists.

The project consists of HTML, CSS and a lot of Javascript since the only data given was to json-files, the first containing the students and the other containing a list of all known pure-blood and half-blod wizards.

The were quite a few customer-requirements such as what should be included in the interface e.g. showing the students, sorting as well as filtering. The interface also had to provide a "popup" window with detailed information about each student, including photo and house-crest and colors.

Furthermore the administrators also had to be able to elect students prefects, make them part og the inquisitorial squad or even expell them.

Tip: When exploring the project, try clicking on the Hogwarts crest.
